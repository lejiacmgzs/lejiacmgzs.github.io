# 乐加传媒工作室官网

## 项目概述

- **项目名称**：乐加传媒工作室官方网站
- **项目描述**：乐加传媒工作室的官方网站，展示公司信息、服务项目和官方公告
- **成立时间**：2024年8月3日
- **许可证号**：LECP-246110202509209624
- **官方网站**：https://lejiacm.studio

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4

## 主要功能

### 1. 首页/Hero 区域
- 品牌 Logo 展示
- 品牌标语："乐在其中，加添人生无限可能！"
- 引导按钮

### 2. 官方公告
- 重要通知（品牌官网、客户服务中心、随悦科技、小陌楠代言人等上线公告）
- 其他通知（合作招募、微信公众号等）

### 3. 关于我们
- 公司介绍
- 品牌理念（专业服务、用心服务、持续创新）
- 成立时间和业务范围

### 4. 服务项目
- 计算机系统维护
- 技术支持
- 定制服务

### 5. 联系方式
- 联系方式展示区域

### 6. 页脚
- 版权信息
- 快速链接
- 许可证号

## 页面结构

```
├── 导航栏 (Header)
│   ├── Logo
│   ├── 导航菜单
│   └── 移动端菜单
├── Hero 区域
├── 官方公告
├── 关于我们
├── 服务项目
├── 联系我们
└── 页脚 (Footer)
```

## 开发命令

- **开发环境**：`pnpm dev`
- **构建**：`pnpm build`
- **生产环境**：`pnpm start`

## 注意事项

1. Logo 图片已下载保存至 `/public/lejia-logo.png`
2. 页面采用响应式设计，支持移动端和桌面端
3. 使用平滑滚动导航
4. 颜色主题以橙色为主色调
